# Contact Management System
Week 3 – Functions & Dictionaries

## Project Description
The Contact Management System is a Python console-based application that allows users to store and manage contact details using dictionaries and functions. The system supports adding, searching, updating, deleting, and displaying contacts with proper input validation and error handling. Contact data is saved using JSON for persistence.

---

## Objectives
- Use functions to organize code
- Store structured data using dictionaries
- Apply string methods for cleaning and formatting
- Implement CRUD operations
- Use file handling for data persistence
- Build a menu-driven application

---

## Features
✔ Add new contacts with validation  
✔ Search contacts using partial name matching  
✔ Update existing contact details  
✔ Delete contacts with confirmation  
✔ Display all contacts  
✔ Save and load contacts from JSON file  
✔ Export contacts to CSV  
✔ View contact statistics  
✔ Error handling for invalid inputs  

---

## Technologies Used
- Python 3
- JSON
- CSV
- Regular Expressions (re)
- Datetime module
